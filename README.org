* Guix User Repository

Personal [[https://guix.gnu.org][GNU Guix]] channel with custom packages not yet available in the official Guix repository.

** Usage

Add this channel to =~/.config/guix/channels.scm=:

#+begin_src scheme
(append
  (list
    (channel
      (name 'guix-user-repository)
      (url "https://github.com/Arthur-Gusmao/GUR")
      (branch "main")))
  %default-channels)
#+end_src

Then run:

#+begin_src bash
guix pull
#+end_src
